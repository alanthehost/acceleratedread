Renaissance Place ID

Renaissance Place ID

What is it? Renaissance Place ID is a unique identifier that provides a shortcut to your Renaissance Place site.

Where can I use it? Renaissance Place ID can be used during setup of Accelerated Reader on iPad®, iPhone®, or iPod touch® and during setup of STAR Reading, STAR Maths and STAR Early Literacy on iPad®.

What if I don’t use Accelerated Reader or STAR assessments? Renaissance Place ID automatically appears on the Renaissance Place Welcome page and can be used wherever you see the RPID image going forward.
